برنامه ای بنویسید که یک عدد بزرگتر از 0 خوانده، سپس به تعداد آن عدد، اعداد دیگری را خوانده، مجموع و میانگین آنها را نمایش دهد.

# مثال

### ورودی

```
4
1 2 3 4 
```

### خروجی

```
10
2.50
```

به فرمت ورودی و خروجی دقت کنید.

### ورودی

```
count 
numbers...
```

### خروجی

```
sum
average
```

* نکته 
- میانگین حتما با ۲ رقم اعشار نمایش داده شود

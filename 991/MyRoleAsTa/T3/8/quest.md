برنامه‌ای بنویسی که عددی از ورودی دریافت و مشخص کند آیا این عدد شرایط زیر را دارد یا خیر.

- فرد باشد
- بزرگتر از 1 و کوچکتر از صد
- رقم یکان آن ۲ باشد


# مثال

### ورودی

```
42
```

### خروجی

```
no
```

به فرمت ورودی و خروجی دقت کنید.

### ورودی

```
number
```

### خروجی

```
yes|no
```


* نکته 
- همه شرایط را باید دارا باشد.

برنامه‌ای بنویسید که ۳ عدد از ورودی دریافت کرده و بنابرحالت عدد سوم یکی از اعمال زیر را انجام دهد.

- اگر مثبت بود حاصل ضرب دو عدد
- اگر صفر بود حاصل جمع دو عدد
- اگر منفی بود تفاضل دو عدد

## مثال

### ورودی
```
12 2 0
```
### خروجی
```
14
```

### فرمت ورودی
```
num1 num2 num3
```
### فرمت خروجی
```
number
```
